# Projeto-To-Do-List

Criado com excelência, este sistema é uma lista onde você pode anotar suas tarefas diárias. Totalmente desenvolvido com React e Vite. Projeto desenvolvido com apoio e acompanhamento das aulas de React do curso Rocketseat. Com intuito teórico e prático.

---

![toDolist](https://github.com/user-attachments/assets/b9b9e462-bce7-41ec-84e9-4c121a7a3556)

## Funcionalidades

- **Listar suas tarefas:** Você pode listar suas atividades, marcar, desmarcar, atualizar e deletar etc.
- **Histórico de operações salvo:** Você terá seu histórico de operações salvo graças ao localStorage.
- **Responsividade:** Projeto altamente responsivo.

## Tecnologias Utilizadas

- **Front-end:** React.js, Vite, Tailwind CSS, TypeScript e localStorage.

## Como Acessar e Rodar

O projeto está hospedado e pode ser acessado diretamente através do navegador:

🔗 [Acessar o projeto online](https://mayconsantos04.github.io/Projeto-To-Do-List/)

### Execução Local

Caso queira baixar o projeto para analisar o código localmente, siga os passos abaixo:

1. Clone o repositório:

```bash
git clone https://github.com/MayconSantos04/Projeto-To-Do-List.git

2. Navegue até a pasta do projeto:

   ```bash
   cd Projeto-To-Do-List
   ```

3. Digite `code .` no terminal para acessar o projeto via VS Code.

4. Instale as dependências e rode a aplicação com Vite:

- npm install
- npm run dev
